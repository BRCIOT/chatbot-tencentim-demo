## 开发环境要求
* HBuilderX 需要升级到最新版本
* TypeScript / JavaScript （TUIKit 使用 ts 语言开发，支持在 js 或者 ts 项目中集成）
* Vue2 / Vue3
* sass（sass-loader 版本 ≤ 10.1.1）
* node（12.13.0 ≤ node , 推荐使用 Node.js 官方 LTS 版本 16.17.0）
* npm（版本请与 node 版本匹配）
